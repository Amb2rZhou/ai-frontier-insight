**AI Frontier Insight Bot 更新日志 v2.0**

**1. 发送格式优化**
接入 Red Lobi 文档能力，日报/周报自动转换为 Redoc 文档链接推送，取代群内长文本，减少刷屏感，阅读体验更好。

**2. Benchmark 排行榜追踪**
新增 AI 模型排行榜自动监控，榜首更替、重要模型上榜等关键变动会自动纳入日报。
- 已接入：Open LLM Leaderboard、SWE-bench Verified
- 规划中：LMSYS Chatbot Arena、MTEB Embedding、VBench 视频生成

**3. 信息来源标注**
每条信号现在都会标注信息的原始来源和权威性——论文标注作者团队/机构，推文标注发推者身份和互动量，GitHub 标注 star 数和组织。帮助大家快速判断信息可信度。

**4. 洞察更 sharp**
Insight 和 Implication 做了大幅精简，每条控制在一句话以内，直接给判断和建议，不再铺垫和绕弯。

欢迎大家继续提出需求，功能持续进化中～

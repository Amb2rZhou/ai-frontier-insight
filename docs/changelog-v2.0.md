**AI Frontier Insight Bot 更新日志 v2.0**

**1. 发送格式优化**
日报/周报改为 GitHub Pages 网页链接推送，不再群内刷屏，阅读体验更好。

**2. Benchmark 排行榜追踪**
新增 5 大 AI 排行榜自动监控，榜首更替、重要模型上榜等关键变动会自动纳入日报：
- Open LLM Leaderboard — 开源模型综合能力
- SWE-bench Verified — 代码工程能力
- ARC-AGI-2 — 抽象推理能力
- OSWorld — 电脑操控能力
- Terminal-Bench 2.0 — 终端编程能力

**3. 信息来源标注**
每条信号标注原始来源和权威性——论文标注机构，推文标注发布者身份和互动量，GitHub 标注 star 数和组织，帮助快速判断可信度。

**4. 洞察更 sharp**
Insight 和 Implication 大幅精简，每条一句话，直接给判断和建议。

欢迎大家继续提出需求，功能持续进化中～

**AI Frontier Insight Bot 更新日志 v2.0**

**1. 发送格式优化**
日报/周报改为 GitHub Pages 网页链接推送，不再群内刷屏，阅读体验更好。

**2. Benchmark 排行榜追踪**
新增 5 大 AI 排行榜自动监控，榜首更替、重要模型上榜等关键变动会自动纳入日报：
- Open LLM Leaderboard — 开源模型综合能力
- SWE-bench Verified — 代码工程能力
- ARC-AGI-2 — 抽象推理能力
- OSWorld — 电脑操控能力
- Terminal-Bench 2.0 — 终端编程能力

**3. 信号筛选更精准**
优化了筛选逻辑，过滤掉非核心话题，论文只收录知名机构和研究者的成果，Twitter 区分专家原创与转发搬运，整体信噪比更高。

**4. 洞察更 sharp、更贴近业务**
Insight 和 Implication 大幅精简，每条一句话，直接给判断和建议。Implication 不再泛泛而谈，会从内容平台、电商平台、推荐系统、创作者工具等具体业务角色出发，给出更贴近实际的建议。

欢迎大家继续提出需求，功能持续进化中～

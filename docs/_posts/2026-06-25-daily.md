### Signal 3: [阿里 Qwen 发布 AgentWorld：不训练做 agent，而是预测环境会返回什么](https://venturebeat.com/technology/alibabas-model-never-trained-as-an-agent-and-improved-agent-performance-across-seven-benchmarks)
*Source: VentureBeat*

阿里 Qwen 团队发布 Qwen-AgentWorld，两款模型不在 agent 环境里直接行动，而是预测环境的返回结果，单一架构覆盖 MCP、Search、Terminal、软件工程、Android、Web、OS 七个领域，并提升了 agent 在这七个基准上的表现。

> 💡 用「世界模型」预测环境反馈，为 agent 训练与评测提供更便宜的模拟器。
> → 环境模拟正成 agent 能力的关键底座，值得关注 agent 仿真/合成数据这一早期方向。

### Signal 4: [小米 HarnessX：让 agent 在任务中自我改写脚手架，小模型受益最大](https://venturebeat.com/orchestration/xiaomis-harnessx-rewrites-its-own-ai-scaffolding-mid-task-and-smaller-models-gain-the-most)
*Source: VentureBeat*

小米研究者提出 HarnessX，针对长时程 agent 任务中「harness（连接 LLM 与环境的软件脚手架）静态、靠人工调」的瓶颈，让系统在执行中依据运行数据自动改写脚手架；实验显示模型越小、提升越明显。

> 💡 agent 性能瓶颈从模型转向脚手架，自改进 harness 把小模型抬到可用线。
> → 自优化的工作流编排成新护城河，利好做 agent 运行时与可观测/自调优工具层的团队。

## Frontier Trend Summary

今日三条主线交叉：一是算力自研与开源同时压成本——OpenAI 端出自研推理芯片 Jalapeño 主打每瓦性能，SambaNova 拟以 100 亿美元估值募资、力推推理芯片做 Nvidia 的低价补充，GLM-5.2 又创开源 ARC-AGI-2 最佳成绩、以接近 Opus 的能力低价拉低前沿实验室利润，闭源性价比护城河持续被侵蚀（闭源-开源差距仍约 8-12 月）。二是 agent 底座从「调模型」转向「调环境与脚手架」——阿里 Qwen-AgentWorld 用世界模型预测七类环境的返回、小米 HarnessX 让 agent 任务中自改写脚手架且小模型受益最大，Mistral OCR 4 把文档抽取升级为结构化版式理解、做企业 RAG 入口。三是资本与叙事换挡——a16z 以 2 亿美元种子押注「会做 AI 研发的 AI」，Qualcomm 收编 Modular 引发开源社区对中立性的担忧，Runpod 拒绝 5 亿美元以上收购；与此同时 SignalFire 数据显示工程岗不降反升，给「AI 取代程序员」叙事降温。

### Signal 4: [Rotary GPU: A New Architecture for Running Large MoE Models Locally with Limited VRAM](https://arxiv.org/abs/2605.29135)
*Source: Arxiv 论文*

一篇来自 arXiv 的论文（Arxiv 2605.29135）提出了 Rotary GPU 架构，通过一种新颖的旋转执行方案，使得在有限 VRAM 的消费级 GPU 上也能运行大规模的 Mixture-of-Experts (MoE) 模型。该方案通过将专家网络动态地加载到 GPU 上执行，克服了传统显存瓶颈，对端侧AI和低成本推理场景有重要意义。虽然来源机构不够明确（未在优先列表中，但内容有启发性），signal_strength 上限为0.70。

> 💡 消费级GPU运行大规模MoE模型成为可能，端侧AI推理成本将显著下降。
> → 社区/内容平台建议探索将部分内容理解、图像标签模型迁移至用户端运行，降低中心推理成本。

### Signal 6: [AV2 Video Standard v1.0 Final Specification Released](https://av2.aomedia.org)
*Source: Hacker News*

开放媒体联盟（AOM）发布了 AV2 视频编码标准的最终 v1.0 规范。作为 AV1 的继任者，AV2 预计将带来更高的压缩效率和更好的图像质量。这一进展对视频内容平台具有长远影响，意味着未来视频流媒体在同等码率下能提供更清晰的画质，或降低带宽成本，对小红书以视频和图文为核心的内容生态构成技术基础升级信号。

> 💡 AV2视频编码标准落地将重塑视频平台带宽成本与画质天花板。
> → 内容平台建议提前评估AV2兼容性，在UGC视频上传转码管线中预留编码升级通道。

### Signal 7: [ADHD: Tree-of-Thought Coding Agent for Claude & Codex Gains Traction](https://github.com/UditAkhourii/adhd)
*Source: GitHub 项目*

开源项目 ADHD（604 stars）是一个基于 Claude 和 Codex Agent SDK 的编码 agent，它实现了带剪枝的“思维树”(Tree-of-Thought) 策略。它能并行发散出不同认知框架下的思路，自动评分并剪除无效路径，最终深化幸存方案。该项目专为创造性和跨学科工作设计，代表了 AI Agent 推理能力在编程领域的实用化趋势。

> 💡 树状推理在编码Agent中的落地证明，复杂任务拆解+剪枝是可复用的Agent能力框架。
> → 内容平台的创意工具Agent（如图像描述生成、文案迭代）建议引入思维树策略提升输出多样性。
